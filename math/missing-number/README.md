# Missing Number

**Источник:** Solvit

## Условие

Дан массив `nums`, содержащий `n` различных чисел из диапазона `[0, n]`. Вернуть единственное число из этого диапазона, которого не хватает в массиве.

## Ограничения

- Массив `nums` содержит `n` различных чисел из диапазона `[0, n]`

## Мой подход

Формула суммы арифметической прогрессии (формула Гаусса), без сортировки:

- `expected` — сумма всех чисел от `0` до `n` включительно, считается по формуле `n * (n + 1) / 2`.
- `real` — фактическая сумма элементов массива.
- Раз в массиве ровно одно число из диапазона `[0, n]` отсутствует, а остальные различны, разница `expected - real` и есть пропущенное число.

## Сложность

- Время: O(n) — один проход по массиву на подсчёт суммы
- Память: O(1)
