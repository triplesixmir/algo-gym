# Plus One

**Источник:** LeetCode #66

## Условие

Большое целое число представлено массивом цифр `digits`, где каждый `digits[i]` — это i-я цифра числа, от старшего разряда к младшему. Число не содержит ведущих нулей. Нужно увеличить это число на единицу и вернуть результат в виде массива цифр.

## Ограничения

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- `digits` не содержит ведущих нулей

## Мой подход

Работаю с массивом напрямую, без перевода в `Number` — при большом количестве цифр (`digits.length` до 100) число не влезет в `Number` без потери точности.

Иду с конца массива к началу:

- Если `digits[i] + 1 !== 10` — переноса разряда нет, прибавляю единицу и сразу возвращаю массив.
- Если получилось 10 — обнуляю текущий разряд и переношу единицу в следующий (более старший) разряд.
- Если перенос дошёл до самого начала массива (`i === 0`) — значит все разряды были девятками (например, `[9,9,9]` → `[0,0,0]`), добавляю `1` в начало через `unshift`.

## Сложность

- Время: O(n) — один проход по массиву с конца
- Память: O(1) — модификация на месте (не считая возможного `unshift`, сдвигающего элементы при переполнении)
