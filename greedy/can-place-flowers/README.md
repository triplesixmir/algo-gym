# Can Place Flowers

**Источник:** LeetCode #605

## Условие

Дана клумба (`flowerbed`) — массив из `0` и `1`, где `0` — пустой участок, `1` — занятый. Цветы нельзя сажать на соседние участки. Дано число `n`. Вернуть `true`, если можно посадить `n` новых цветов, не нарушив правило "не рядом", иначе `false`.

## Ограничения

- `1 <= flowerbed.length <= 2 * 10^4`
- `flowerbed[i]` — `0` или `1`
- В исходной клумбе нет двух соседних цветов
- `0 <= n <= flowerbed.length`

## Мой подход

Жадный проход по массиву слева направо: на каждой позиции `i` проверяю, свободна ли она сама и оба её соседа (`i-1` и `i+1`). Если да — сажаю цветок и сразу помечаю позицию как занятую (`flowerbed[i] = 1`), чтобы это учлось при проверке следующей позиции.

Отдельно обрабатываю границы массива: для позиций `i-1` или `i+1`, выходящих за пределы массива, подставляю `0` — по смыслу задачи "за пределами клумбы" эквивалентно "там пусто", то есть не мешает посадке с этой стороны.

В конце сравниваю посчитанное количество посаженных цветов `k` с `n`.

## Сложность

- Время: O(n) — один проход по массиву
- Память: O(1) — модификация на месте, без дополнительных структур
