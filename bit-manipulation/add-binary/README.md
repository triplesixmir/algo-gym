# Add Binary

**Источник:** LeetCode #67

## Условие

Даны две двоичные строки `a` и `b`. Вернуть их сумму в виде двоичной строки.

## Ограничения

- `1 <= a.length, b.length <= 10^4`
- `a` и `b` состоят только из символов `'0'` или `'1'`
- Строки не содержат ведущих нулей, кроме случая, когда число само равно нулю

## Мой подход

Сложение в столбик справа налево, как обычное сложение чисел на бумаге, только с основанием 2 вместо 10:

- Два указателя `i` и `j` идут с конца строк `a` и `b` к началу.
- На каждом шаге беру перенос (`carry`) из предыдущего разряда и прибавляю к нему текущие цифры `a` и `b` (если строка ещё не закончилась) — получаю `sum` (от 0 до 3).
- Текущая цифра результата — `sum % 2` (остаток от деления на 2), новый перенос — `Math.floor(sum / 2)` (целая часть).
- Собираю результат, добавляя новую цифру **в начало** строки, потому что идём от младшего разряда к старшему.
- Цикл продолжается, пока не закончились обе строки **и** не обнулился перенос — если после обеих строк остался перенос (например, `1 + 1`), он допишется отдельным разрядом.

Строки при этом не переводятся в `Number` — при длине до 10^4 символов число не влезло бы в стандартный числовой тип без потери точности.

## Сложность

- Время: O(max(n, m)) — один проход по более длинной из строк
- Память: O(max(n, m)) — под результат
