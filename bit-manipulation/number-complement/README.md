# Number Complement

**Источник:** LeetCode #476

## Условие

Дополнение целого числа — это число, получаемое инвертированием всех битов его двоичного представления (`0` → `1`, `1` → `0`). Дано целое число `num`, вернуть его дополнение.

## Ограничения

- `1 <= num < 2^31`

## Мой подход

- Перевожу `num` в двоичную строку (`toString(2)`) и разбиваю на массив символов.
- Прохожу по каждому символу: если это `'1'` — кладу в результат `0`, иначе — `1`. Так получаю инвертированную двоичную последовательность.
- Собираю обратно в строку и парсю её как двоичное число через `parseInt(..., 2)`, получая итоговый десятичный результат.

## Сложность

- Время: O(log num) — количество битов в числе
- Память: O(log num) — под массив и строку
