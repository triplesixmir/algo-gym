# Is Subsequence

**Источник:** LeetCode #392

## Условие

Даны строки `s` и `t`. Вернуть `true`, если `s` является подпоследовательностью `t`, иначе `false`. Подпоследовательность — строка, полученная из исходной удалением некоторых (возможно, ни одного) символов без изменения относительного порядка оставшихся.

## Ограничения

- `0 <= s.length <= 100`
- `0 <= t.length <= 10^4`
- `s` и `t` состоят только из строчных латинских букв

Follow-up: если приходит очень много разных `s` (`k >= 10^9`) для одного и того же `t`, как изменить решение?

## Мой подход

Два указателя:

- `left` идёт по `s`, `right` — по `t`.
- Если `s[left] === t[right]` — нашли очередной нужный символ `s` в `t`, сдвигаю `left`.
- `right` двигается на каждом шаге независимо от совпадения — можно пропускать "лишние" символы `t`.
- Если `left` дошёл до конца `s` — все символы `s` нашлись в `t` в правильном порядке, значит `s` — подпоследовательность `t`.

**Про follow-up:** текущее решение при каждом вызове заново проходит `t` целиком — при огромном числе запросов (`k >= 10^9`) это станет узким местом. Правильный подход — один раз предпосчитать по `t` структуру (например, для каждой буквы алфавита — отсортированный список индексов, где она встречается), а на каждый запрос использовать бинарный поиск по этим индексам вместо линейного прохода.

## Сложность

- Время: O(n + m), где n = s.length, m = t.length
- Память: O(1)
