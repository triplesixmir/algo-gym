# Valid Anagram

**Источник:** LeetCode #242

## Условие

Даны две строки `s` и `t`. Вернуть `true`, если `t` — анаграмма `s`, иначе `false`.

## Ограничения

- `1 <= s.length, t.length <= 5 * 10^4`
- `s` и `t` состоят только из строчных латинских букв

## Мой подход

Подсчёт частоты символов через два `Map`:

- Считаю частоту каждого символа в `s` (`sMap`) и в `t` (`tMap`).
- Быстрый отсев: если количество **уникальных** символов не совпадает (`sMap.size !== tMap.size`) — точно не анаграммы.
- Иначе прохожу по всем парам `[key, value]` из `sMap` и сверяю частоту каждого символа с `tMap.get(key)`. Если хоть один не совпал — не анаграмма.

**Альтернатива:** более короткое решение через сортировку:

```js
var isAnagram = function(s, t) {
    return s.split("").sort().join("") === t.split("").sort().join("");
};
```

Идея: если строки — анаграммы, то отсортированные посимвольно версии совпадут полностью. Компактнее по коду, но даёт O(n log n) вместо O(n) из-за сортировки, поэтому основным выбран вариант с `Map`.

## Сложность

- Время: O(n + m), где n и m — длины `s` и `t`
- Память: O(n + m) — под два `Map`
