# Reverse Vowels Of A String

**Источник:** LeetCode #345

## Условие

Дана строка `s`. Развернуть только гласные буквы в строке (порядок остальных символов не менять) и вернуть результат. Гласные — `'a', 'e', 'i', 'o', 'u'` в любом регистре, могут встречаться несколько раз.

## Ограничения

- `1 <= s.length <= 3 * 10^5`
- `s` состоит из печатных ASCII-символов

## Мой подход

Два указателя, сходящиеся к центру строки:

- `Set` с гласными буквами (обе регистра) для проверки за O(1).
- Строки в JS иммутабельны, поэтому перевожу `s` в массив символов (`split('')`), чтобы менять элементы на месте, а в конце собираю обратно (`join('')`).
- `left` идёт с начала, `right` — с конца. Внутренние `while`-циклы пропускают не-гласные символы с каждой стороны (с проверкой `left < right`, чтобы не потерять границы).
- Когда оба указателя остановились на гласных — меняю их местами через деструктуризацию, затем сдвигаю оба указателя к центру.

## Сложность

- Время: O(n) — каждый символ обрабатывается не более пары раз
- Память: O(n) — под массив символов
