# Greatest Common Divisor Of Strings

**Источник:** LeetCode #1071

## Условие

Для двух строк `s` и `t` говорят, что "`t` делит `s`", если `s` можно получить, повторив `t` один или несколько раз подряд (`s = t + t + ... + t`). Даны строки `str1` и `str2`, нужно вернуть самую длинную строку `x`, которая делит одновременно и `str1`, и `str2`.

## Ограничения

- `1 <= str1.length, str2.length <= 1000`
- `str1` и `str2` состоят из заглавных латинских букв

## Мой подход

Аналог поиска НОД (наибольшего общего делителя), только для строк:

- **Проверка существования общего делителя:** если у `str1` и `str2` есть общий "строительный блок" `x`, то обе строки — это просто разное число повторений `x`, и тогда `str1 + str2 === str2 + str1` (порядок склейки не важен, получается одна и та же последовательность повторов). Если равенство не выполняется — общего делителя не существует, возвращаю `""`.
- **Поиск длины `x`:** раз обе строки состоят из целого числа копий `x`, длина `x` обязана быть общим делителем длин `str1` и `str2`. Ищу наибольшее такое число перебором от `min(len1, len2)` вниз до тех пор, пока не найду число, на которое обе длины делятся нацело.
- Ответ — префикс `str1` (или `str2`, они совпадут) длиной, равной найденному НОД длин.

## Сложность

- Время: O(n + m) на проверку конкатенации + O(min(n, m)) на поиск НОД длин
- Память: O(n + m) — под склеенные строки при проверке
