# Length Of Last Word

**Источник:** LeetCode #58

## Условие

Дана строка `s`, состоящая из слов и пробелов. Вернуть длину последнего слова в строке. Словом считается максимальная подстрока, состоящая только из символов, отличных от пробела.

## Ограничения

- `1 <= s.length <= 10^4`
- `s` состоит только из английских букв и пробелов `' '`
- В строке гарантированно есть хотя бы одно слово

## Мой подход

- `s.trim()` убирает пробелы по краям строки.
- `split(/\s+/)` разбивает строку на слова по любому количеству пробелов подряд.
- Последний элемент полученного массива — последнее слово, возвращаю его длину.

## Сложность

- Время: O(n) — один проход по строке при `trim`/`split`
- Память: O(n) — `split` создаёт новый массив слов
