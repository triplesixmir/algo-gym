# Merge Strings Alternately

**Источник:** LeetCode #1768

## Условие

Даны две строки `word1` и `word2`. Слить их, чередуя буквы, начиная с `word1`. Если одна строка длиннее другой, оставшиеся буквы дописываются в конец результата.

## Ограничения

- `1 <= word1.length, word2.length <= 100`
- `word1` и `word2` состоят только из строчных латинских букв

## Мой подход

Один проход по индексу `i` от `0` до `Math.max(word1.length, word2.length)`:

- Если у `word1` ещё есть символ на позиции `i` — добавляю его в результат.
- Если у `word2` ещё есть символ на позиции `i` — добавляю и его.
- Так автоматически покрывается случай, когда одна из строк короче другой: как только она заканчивается, добавляются только символы более длинной строки.

## Сложность

- Время: O(n + m), где n и m — длины `word1` и `word2`
- Память: O(n + m) — под итоговую строку
