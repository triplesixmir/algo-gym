# Convert Sorted Array To Binary Search Tree

**Источник:** LeetCode #108

## Условие

Дан массив целых чисел `nums`, отсортированный по возрастанию. Преобразовать его в сбалансированное по высоте бинарное дерево поиска (height-balanced BST).

## Ограничения

- Элементы массива уникальны, отсортированы по возрастанию

## Мой подход

Рекурсивное построение дерева "разделяй и властвуй":

- Вспомогательная функция `convert(left, right)` строит поддерево из куска массива `nums[left..right]`.
- Базовый случай: `left > right` — кусок массива пустой, узла тут нет, возвращаю `null`.
- Беру средний индекс `mid` этого куска и создаю из `nums[mid]` узел — средний элемент отсортированного диапазона делит его на две примерно равные половины, что и обеспечивает сбалансированность дерева.
- Рекурсивно строю левое поддерево из части массива до `mid` (`convert(left, mid - 1)`) и правое поддерево из части после `mid` (`convert(mid + 1, right)`).
- Итоговое дерево строится через `convert(0, nums.length - 1)` — весь массив целиком.

## Сложность

- Время: O(n) — каждый элемент массива становится узлом дерева ровно один раз
- Память: O(n) — под узлы дерева, плюс O(log n) на стек рекурсии (при сбалансированном дереве)
