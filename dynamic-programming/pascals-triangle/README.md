# Pascal's Triangle

**Источник:** LeetCode #118

## Условие

Дано целое число `numRows`. Вернуть первые `numRows` рядов треугольника Паскаля. Каждое число в треугольнике — сумма двух чисел прямо над ним.

## Ограничения

- `1 <= numRows <= 30`

## Мой подход

Рекурсивное построение "снизу вверх" через предыдущий результат:

- `generate(numRows)` сначала рекурсивно строит все предыдущие ряды через `generate(numRows - 1)`.
- Базовый случай: `numRows === 1` — треугольник из одного ряда `[[1]]`.
- Новый (последний) ряд создаю длиной `numRows`, изначально заполненный единицами — крайние элементы каждого ряда треугольника Паскаля всегда равны `1`.
- Внутренние значения нового ряда считаю как сумму двух чисел прямо над ними — беру их из последнего ряда `prevRows` (индекс `numRows - 2`, так как в `prevRows` уже `numRows - 1` рядов).
- Добавляю новый ряд в конец `prevRows` и возвращаю весь треугольник.

## Сложность

- Время: O(numRows²) — суммарное количество элементов во всех рядах треугольника растёт квадратично
- Память: O(numRows²) — под сам треугольник, плюс O(numRows) на стек рекурсии
