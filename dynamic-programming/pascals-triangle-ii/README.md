# Pascal's Triangle II

**Источник:** LeetCode #119

## Условие

Дано целое число `rowIndex`. Вернуть `rowIndex`-й (нумерация с нуля) ряд треугольника Паскаля.

## Ограничения

- `0 <= rowIndex <= 33`

## Мой подход

Итеративное построение, ряд за рядом, с хранением только текущего ряда (в отличие от Pascal's Triangle I, где хранится весь треугольник):

- Стартую с `res = [1]` — нулевой ряд.
- На каждой итерации пересчитываю `res` через `map`: новое значение на позиции `index` — сумма `res[index - 1]` и `res[index]` **из старого** ряда (JS полностью вычисляет `map(...)`, прежде чем переприсвоить результат в `res`, поэтому внутри колбэка `res` всё ещё указывает на предыдущий, ещё не изменённый ряд).
- `res[index - 1] || 0` — обрабатывает крайний левый случай, где "слева от начала ряда" ничего нет.
- После пересчёта середины дописываю замыкающую `1` через `push` — правый край нового ряда.
- Повторяю `rowIndex` раз, получая нужный ряд.

## Сложность

- Время: O(rowIndex²) — суммарная работа по всем рядам
- Память: O(rowIndex) — хранится только текущий ряд, а не весь треугольник
