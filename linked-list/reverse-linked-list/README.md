# Reverse Linked List

**Источник:** LeetCode #206

## Условие

Дан `head` односвязного списка. Развернуть список и вернуть новый `head`.

## Ограничения

- Количество узлов в списке — от 0 до 5000
- `-5000 <= Node.val <= 5000`

## Мой подход

Итеративный разворот списка тремя указателями:

- `node` — уже развёрнутая часть списка (изначально `null`), `head` — текущий узел, который ещё предстоит обработать.
- На каждом шаге сначала сохраняю `head.next` в `next` (иначе потеряю остаток списка при развороте ссылки).
- Разворачиваю ссылку: `head.next = node` — текущий узел теперь указывает на уже развёрнутую часть.
- Сдвигаю оба указателя вперёд: `node = head`, `head = next`.
- Когда `head` становится `null` — весь список развёрнут, `node` указывает на новую голову.

Пустой список (`head = null` изначально) обрабатывается автоматически: цикл не выполняется, сразу возвращается `null`.

## Сложность

- Время: O(n) — один проход по списку
- Память: O(1) — разворот на месте, без создания новых узлов
