# Remove Linked List Elements

**Источник:** LeetCode #203

## Условие

Дан `head` односвязного списка и целое число `val`. Удалить все узлы, у которых `Node.val === val`, и вернуть новую голову списка.

## Ограничения

- Количество узлов в списке — от 0 до 10^4
- `1 <= Node.val <= 50`
- `0 <= val <= 50`

## Мой подход

Фиктивный (dummy) узел перед головой списка — стандартный приём, чтобы не обрабатывать удаление самой головы отдельным случаем:

- `res` — фиктивный узел, указывающий на настоящую голову. Если нужно удалить сам первый элемент списка, отдельная ветка не нужна — просто меняется `res.next`.
- Внешний `while (dummy)` идёт по списку.
- Внутренний `while (dummy.next && dummy.next.val === val)` "выкидывает" все подряд идущие совпадения, прежде чем сдвинуться дальше — это важно, иначе движение вперёд после каждого удаления пропустило бы следующий узел, если он тоже совпадает.
- В конце возвращаю `res.next`, а не исходный `head` — потому что сама голова могла быть удалена.

## Сложность

- Время: O(n) — один проход по списку
- Память: O(1) — модификация списка на месте, без создания новых узлов (кроме одного фиктивного)
