# Remove Duplicates From Sorted List

**Источник:** LeetCode #83

## Условие

Дан `head` отсортированного связного списка. Удалить все дубликаты так, чтобы каждый элемент встречался только один раз. Вернуть список отсортированным.

## Ограничения

- Количество узлов в списке — от 0 до 300
- `-100 <= Node.val <= 100`
- Список гарантированно отсортирован по возрастанию

## Мой подход

Список отсортирован, значит все дубликаты идут подряд — достаточно сравнивать каждый узел только с соседним:

- `current` идёт по списку, пока сам он и `current.next` не равны `null`.
- Если `current.next.val === current.val` — следующий узел дубликат, "выкидываю" его, перепривязывая `current.next` сразу к `current.next.next`. При этом `current` не двигаю — вдруг за удалённым узлом идёт ещё один дубликат того же значения, его тоже нужно проверить на этом же шаге.
- Если значения разные — сдвигаю `current` на следующий узел.

## Сложность

- Время: O(n) — один проход по списку
- Память: O(1) — модификация списка на месте, без создания новых узлов
