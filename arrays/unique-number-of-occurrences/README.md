# Unique Number Of Occurrences

**Источник:** LeetCode #1207

## Условие

Дан массив целых чисел `arr`. Вернуть `true`, если количество вхождений каждого значения в массиве уникально, иначе `false`.

## Ограничения

- `1 <= arr.length <= 1000`
- `-1000 <= arr[i] <= 1000`

## Мой подход

- Считаю частоту каждого значения с помощью `Map`: ключ — значение из массива, значение — сколько раз оно встретилось.
- Строю `Set` из всех подсчитанных частот (`freq.values()`) — если среди частот были повторы, `Set` схлопнет дубликаты, и его размер станет меньше `freq.size`.
- Если размеры `freq` и `val` совпали — значит повторов не было, все частоты уникальны.

## Сложность

- Время: O(n) — один проход на подсчёт частот
- Память: O(n) — под `Map` и `Set`
