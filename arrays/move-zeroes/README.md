# Move Zeroes

**Источник:** LeetCode #283

## Условие

Дан массив целых чисел `nums`. Переместить все нули в конец массива, сохранив относительный порядок ненулевых элементов. Модификация должна выполняться на месте, без создания копии массива.

## Ограничения

- `1 <= nums.length <= 10^4`
- `-2^31 <= nums[i] <= 2^31 - 1`

Follow-up: минимизировать количество операций.

## Мой подход

Two pointers, аналогично Remove Element, только вместо удаления — перестановка:

- `left` — граница, до которой все элементы уже ненулевые и стоят в правильном порядке.
- `right` пробегает весь массив. Как только находит ненулевой элемент — меняет его местами с `nums[left]` через деструктуризацию и сдвигает `left`.
- Если `right` указывает на тот же элемент, что и `left` (ещё не встретилось ни одного нуля) — своп ничего не меняет, элемент просто меняется местами сам с собой.
- Порядок ненулевых элементов сохраняется, потому что элементы не удаляются и не двигаются произвольно — только меняются местами с позицией `left`, которая всегда указывает на следующее "свободное" место среди уже обработанных ненулевых элементов.

## Сложность

- Время: O(n) — один проход по массиву
- Память: O(1) — модификация на месте, без дополнительных структур
