# Maximum Average Subarray I

**Источник:** LeetCode #643

## Условие

Дан массив целых чисел `nums` из `n` элементов и число `k`. Найти непрерывный подмассив длины `k` с максимальным средним значением и вернуть это значение.

## Ограничения

- `n == nums.length`
- `1 <= k <= n <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

## Мой подход

Скользящее окно (sliding window) фиксированного размера `k`:

- Считаю сумму первого окна (первые `k` элементов) обычным циклом — стартовая точка.
- Дальше окно скользит по одному элементу: на каждом шаге вычитаю элемент, который "выпал" слева (`nums[i - k]`), и прибавляю новый элемент справа (`nums[i]`) — обновление суммы окна за O(1), без пересчёта заново.
- По ходу скольжения отслеживаю `maxSum` — максимальную сумму среди всех окон.
- Делить на `k` нужно только один раз, в самом конце: раз `k` одинаково для всех окон, максимум среднего достигается там же, где максимум суммы.

## Сложность

- Время: O(n) — один проход по массиву
- Память: O(1)
