# Find The Difference Of Two Arrays

**Источник:** LeetCode #2215

## Условие

Даны два массива целых чисел `nums1` и `nums2`. Вернуть список `answer` из двух списков:

- `answer[0]` — все различные числа из `nums1`, которых нет в `nums2`.
- `answer[1]` — все различные числа из `nums2`, которых нет в `nums1`.

Порядок чисел внутри списков не важен.

## Ограничения

- `1 <= nums1.length, nums2.length <= 1000`
- `-1000 <= nums1[i], nums2[i] <= 1000`

## Мой подход

Перевожу оба массива в `Set` (это сразу убирает дубликаты и даёт проверку на вхождение за O(1)), а дальше использую встроенный метод `Set.prototype.difference()`:

- `set1.difference(set2)` — элементы, которые есть в `set1`, но отсутствуют в `set2`.
- `set2.difference(set1)` — аналогично в обратную сторону.

`Set.difference()` — относительно новый метод (стандартизирован в 2024), доступен начиная с Node.js 22+ и современных браузеров. Если бы код запускался в среде со старым движком, пришлось бы делать это вручную через `filter` + `has()`.

## Сложность

- Время: O(n + m) — построение множеств и вычисление разностей
- Память: O(n + m) — под множества
