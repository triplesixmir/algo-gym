# Contains Duplicate

**Источник:** LeetCode #217

## Условие

Дан массив целых чисел `nums`. Вернуть `true`, если хотя бы одно значение встречается в массиве два и более раз, и `false`, если все элементы различны.

## Ограничения

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

## Мой подход

`Set` с ранним выходом:

- Иду по массиву, для каждого элемента проверяю `set.has(nums[i])`.
- Если элемента ещё нет — добавляю его в `set`.
- Если уже есть — значит видел его раньше, сразу `return true`, не тратя время на оставшуюся часть массива.
- Если дошёл до конца без совпадений — `return false`.

**Альтернатива:** можно решить и через сортировку — отсортировать массив и проверить соседние элементы на равенство (дубликаты после сортировки всегда оказываются рядом). Но это O(n log n) вместо O(n) и мутирует исходный массив, поэтому вариант с `Set` предпочтительнее.

## Сложность

- Время: O(n) в среднем случае
- Память: O(n) — под `Set`
