# Remove Element

**Источник:** LeetCode #27

## Условие

Даны массив целочисленных значений `nums` и целое число `val`, удалить все вхождения `val` в `nums` на месте (in-place). Порядок элементов можно менять. Функция должна вернуть `k` — количество элементов в `nums`, не равных `val`. При этом первые `k` элементов массива должны содержать элементы, не равные `val`, остальное значения не имеет.

## Ограничения

- `0 <= nums.length <= 100`
- `0 <= nums[i] <= 50`
- `0 <= val <= 100`

## Мой подход

Two pointers (slow/fast):

- `fast` пробегает по всем элементам массива.
- `slow` указывает на позицию, куда нужно записать очередной элемент, не равный `val`.
- Если `nums[fast] !== val`, копируем его на позицию `slow` и сдвигаем `slow` вперёд.
- В конце `slow` — это и есть количество элементов, не равных `val` (то самое `k`), потому что он увеличивался ровно на каждый такой элемент.

## Сложность

- Время: O(n) — один проход по массиву
- Память: O(1) — модификация на месте, без дополнительных структур
