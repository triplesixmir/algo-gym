# Find The Highest Altitude

**Источник:** LeetCode #1732

## Условие

Велосипедист едет по маршруту из `n + 1` точек на разных высотах, начиная с точки `0` на высоте `0`. Дан массив `gain` длины `n`, где `gain[i]` — изменение высоты между точками `i` и `i + 1`. Вернуть максимальную высоту среди всех точек маршрута.

## Ограничения

- `n == gain.length`
- `1 <= n <= 100`
- `-100 <= gain[i] <= 100`

## Мой подход

Накопительная сумма (prefix sum) с отслеживанием текущего максимума:

- `alt` — текущая высота, стартует с `0` (высота стартовой точки).
- `max` тоже стартует с `0` — точка 0 сама по себе уже кандидат на максимум, даже если весь дальнейший путь идёт только вниз.
- На каждом шаге прибавляю `gain[i]` к `alt` (получаю высоту следующей точки) и сравниваю с текущим `max`.

## Сложность

- Время: O(n) — один проход по массиву
- Память: O(1)
