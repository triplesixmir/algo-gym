# Number Of Rectangles That Can Form The Largest Square

**Источник:** LeetCode #1725

## Условие

Дан массив `rectangles`, где `rectangles[i] = [li, wi]` — длина и ширина i-го прямоугольника. Из прямоугольника можно вырезать квадрат со стороной `k`, если `k <= li` и `k <= wi` (то есть максимальная сторона квадрата — `min(li, wi)`). Пусть `maxLen` — сторона самого большого квадрата, который можно получить из любого прямоугольника. Вернуть, сколько прямоугольников позволяют получить квадрат именно такого размера.

## Ограничения

- `1 <= rectangles.length <= 1000`
- `rectangles[i].length == 2`
- `1 <= li, wi <= 10^9`
- `li != wi`

## Мой подход

Один проход по массиву с подсчётом частот через `Map`:

- Для каждого прямоугольника нахожу максимально возможную сторону квадрата — `len = min(li, wi)`.
- По ходу отслеживаю `maxLen` — наибольшее встретившееся значение `len`.
- Параллельно копирую частоту каждого `len` в `Map` (сколько прямоугольников дают квадрат именно такого размера).
- В конце `map.get(maxLen)` — количество прямоугольников, которые дают квадрат максимального размера.

## Сложность

- Время: O(n) — один проход по массиву
- Память: O(n) — под `Map` в худшем случае (если все длины разные)
