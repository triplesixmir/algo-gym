# Sqrt X

**Источник:** LeetCode #69

## Условие

Дано неотрицательное целое число `x`. Вернуть квадратный корень из `x`, округлённый вниз до ближайшего целого. Нельзя использовать встроенные функции возведения в степень (`pow`, `x ** 0.5` и т.п.).

## Ограничения

- `0 <= x <= 2^31 - 1`

## Мой подход

Бинарный поиск по ответу — среди чисел от `1` до `x/2` (для `x >= 2` точный корень никогда не превышает половину числа) ищу наибольшее `mid`, для которого `mid * mid <= x`:

- Если `mid * mid === x` — нашли точный корень, возвращаю сразу.
- Если `mid * mid < x` — корень больше, сдвигаю `left = mid + 1`.
- Если `mid * mid > x` — корень меньше, сдвигаю `right = mid - 1`.
- Когда цикл заканчивается без точного совпадения, `right` — это последнее значение, чей квадрат был меньше `x`, то есть искомый ответ, округлённый вниз.

Отдельно обрабатываю `x < 2` (то есть `0` и `1`) — для них корень равен самому числу.

## Сложность

- Время: O(log x) — бинарный поиск
- Память: O(1)
